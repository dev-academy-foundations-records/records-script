Blocked on a simple problem

I am going to be as honest as possible with the following problem. JavaScript as a whole has been quite difficult to wrap my head around. I will refer to Sprint-3 on JavaScript cafe as an example because it was quite a simple problem that I just could not get my head around. we had to essentailly make an app that hosted food and beverage options for people to choose from. In order to generate an order, it was required from us to be able to write code that would allow us to generate a random order within the parameters set. So for Example if the menu had 4 items in it, the button named 'Generate Order' would be called and theoratically a random order of said menu items would be returned. However I could not for the life of me understand why the GetrandomInt() function would not work.

What problem solving technique did I use?

The first thing I did to help me with this issue was ask for help. I know, I know, I know, I know, I know.... But I waited far too long to do it. Luckly for me in the #Foundations Discord channel I found out about a JavaScript Q & A that was being held by one of the Facilitators. So I took it upon myself to ask 'Gerard' if he could steer me in the right direction, and sure enough he did. 'Gerard' had informed me that the reason the GetRandomInt() function would not work was because it was not defined.
