No seriously.... what does Web Development mean?

Web development refers in general to the tasks associated with developing websites for hosting via intranet or internet. The web development process includes web design, web content development, client-side/sever-side scripting and network security configuration, among other tasks. In a broader sense, web development encompasses all the actions , updates and operations required to build, maintain and manage a website to ensure it's performance, user experience , and speed are optimal/ Web development is also knowns as website development, while the professionals (soon to be professional like myself hehe :)__) that maintain a website are called web developers or Devs.
