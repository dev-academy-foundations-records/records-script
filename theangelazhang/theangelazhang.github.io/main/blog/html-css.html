It can be hard to place CSS elements where we want them to go. Thankfully we have some positioning commands up our sleeve, namely relative, absolute and fixed positioning.

Relative positioning

Allows you to adjust the position of an element relative to its normal position. For example, the code below moves the rectangle 50px horizontally from where it would usually be.



Absolute positioning

Lets you position an element exactly where you want it to be on the page. The values will be relative to the next parent element with relative (or absolute) positioning. If there isn’t a parent, then the positioning will be relative to the html element — i.e. relative to the page itself.



Fixed positioning

Fixes the element to the viewport (the browser window) and doesn’t change when scrolling. Useful for navigation bars that you want to see all the time. The back link to the left is an example of a fixed element.
