Static Positioning

"Ogers are like onions... "

and so is static postioning!
"Why?" I hear you ask... because they all have layers!
.... yes.... cakes have layers too.

Static positioning is simply layers on a website that start from top to bottom. If you put a red layer at the top, nothing else goes into that layer. If you add another layer, it will automatically go underneath the red layer. This all works together to make a very tasty looking webpage... if you like layers ;p

Here is an example of static layers. See the beautiful layered rainbow?

Absolute Positioning

This one, had me seriously confused. Everytime I used absolute, it would just do the same as relative. I found an image on W3School (which is great if you haven't checked it out yet). It shows that an absolute element must have a parent element otherwise it will use the body or HTML as a parent. I tried putting my absolute element uder div parents but it still didn't work.

Finally, I realised that W3School also included the parent whose position was relative.


Now this analogy is a bit of a stretch but bare with me. Imagine a small clingy child who always wants to hold the hand of their parent. If the parent all of a sudden vanishes, the small child, helpless, seeks another hand to hold. (Remember this is an analogy only). This means if there is no parent element, the 'child' element will find the next closest thing.

An issue occurs though if the adult themselves is lost. If they do not know their own position relative to the world (or in this case our DOM), they can't look after their child and the child will seek another 'parent'.

I hope this analogy wasn't too traumatic for anyone.

Fixed Positioning

Imagine you have a sticky note stuck to your screen. If you scroll with your mouse, does it move? No.

An image or other element such as a navigaion bar can have fixed postitioning. This means that even if you scroll, it will stay in the same place. The position of the element is based on your window and will stay in the same spot in your window.

See donkey to the side by the scroll bar? He is an example of a fixed element.
