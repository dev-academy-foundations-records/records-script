What does it mean to display inline vs inline blocks

Created by Kurt 20/01/2023

Both inline and inline-block have the functions to adjust the elements in the layout of the website. While this blog will explain the use of inline and inline-block, and the differences between.

Inline allow the elements stay in the same line, instead of pushing the element to the lines above or under, elements set as inline only take the space of the contents, which means they igonre the height and weidth properties. change the size of viewport will automatically wrap the text with inline value. As we can see, inline value allow the elements sit together in the same line, and only take the space three blocks with inline value were set both height and width 1000px, but it does not show on the Website. When you adjust the viewport, the blocks who take the whole line will be pushed to next line. The contents be pushed to the next line has the height of the Text.

In terms of inline-block, it blends some of the functions from both inline and block. Which means It will display as inline value, allowing elements sit together, but the height and width of the elements are not defaut size of Text.
