We add a little bit of padding, because padding is nice. Also will help us see the difference between, inline and inline-block.



we add inline to our div elements with css



so this is what it looks like when we use inline block, all the div elements are placed on the same line. Notice how the padding width is respected but not the height. you cannot change the height or the width of a div if you use inline.



now we change "inline" to be "inline-block".



Now the padding height is respected as well, now there is no overlap of the block. if you use inline-block you can change the width and the height of the element.



so to summarise, inline block is good for fitting different elements into the same line where yo u dont want to use things like padding or margin or change the width or height,

adding a link onto a paragraph would be an example of somthing that you would use inline for. If you are aranging elements where sizing and spacing is somthing the is important you should use inline-block instead.
